Firebird Database Server 1.0
============================

Этот документ является руководством по установке на платформе Win32.

Содержание
--------

o Перед установкой
o Другие замечания

Перед установкой
-------------------

  ВАЖНО!

  Этот установочный пакет пытается найти установленные или
  работающие версии Firebird или Interbase.

  Перед тем как продолжить установку Вы должны ОСТАНОВИТЬ
  работу сервера Interbase или Firebird, или удалить уже
  установленную версию.


  Остановка сервера

  o Если он запущен как сервис - остановите его из Control Panel.
  o Если он запущен как приложение - просто закройте его.


Другие замечания
-----------

  Для установки Firebird как сервиса требуются административные права.

  Этот установочный скрипт не производит конфигурирование Guardian
  (ibguard.exe).
  На данный момент нет инструментов, позволяющих сконфигурировать Guardian как
  сервис. Эта проблема имеет значение только под Windows NT (в W2K рестарт
  ibserver.exe конфигурируется в сервисах). Для рестарта сервера под NT
  пользователь должен иметь административные права. Последующие варианты
  инсталлятора будут содержать опцию по установке Guardian как сервиса.


Установка из командных файлов
------------------------------

Инсталлятор может быть запущен из командного файла. Это удобно,
когда Вы производите установку клиентской части из сети.
Следующие параметры могут быть переданы:


/SP-
  Убирает вопрос
  This will install... Do you wish to continue?
  в самом начале установки

/SILENT, /VERYSILENT
  Указывает инсталлятору выдавать минимум сообщений или не выдавать их совсем.
  В режиме silent окно инсталлятора не показывается, в то время как
  отображается только прогресс установки. В режиме very silent не показывается
  и прогресс установки. В остальном все как обычно, например показываются
  сообщения об ошибках и задается начальный вопрос установки (если Вы его не
  запретили командой /SP-, указанной выше).

  Если нужно рестартовать компьютер, и команда /NORESTART не была
  указана, инсталлятор все равно покажет сообщение "Reboot now?".
  В режиме very silent этот вопрос не будет задаваться, и рестарт произойдет
  автоматически.


/NORESTART
  Указывает инсталлятору не перезагружать операционную систему
  (если в скрипте инсталляции это было задано).

/DIR="x:\dirname"
  Изменяет каталог по умолчанию, запрашиваемый в окне
  "Select Destination Directory". Должен быть указан полный путь с именем диска
  Если в скрипте инсталляции, в секции [Setup] директива DisableDirPage была
  установлена в yes, то этот параметр командной строки игнорируется.

/GROUP="folder name"
  Изменяет имя папки по умолчанию, отображаемое в окне
  "Select Start Menu Folder". Если в секции [Setup] директива
  DisableProgramGroupPage была установлена в yes, то этот параметр командной
  строки игнорируется.

/NOICONS
  Указывает инсталлятору включить опцию "Don't create any icons"
  в окне Select Start Menu Folder.

/COMPONENTS="component name"

  Выберите из Server, DevTools или Client

  Изменяет умолчательный набор компонент. Использование этого параметра
  командной строки приводит к автоматическому выбору установки "custom type".
